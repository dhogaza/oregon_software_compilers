     std/ldp index limited to -512..504 bytes (9 bit offset * 2).
     if > 504 but less 4096+504
       add ip0, sp, #(4096-504)
       ldp/stp r1, r2, [ip0, 504]
     else if <= $FFFFFF and masked with $E00 = 0
       add ip0, sp, #(index div $1000) shift 12
       ldp/stp r1, r2, [ip0, (index mask $1FF)]
     else
       mov ip0, (index div $FFFF) shift 16
       if index and $FFFF <> 0 then
         movk ip0, index and $FFFF
       ldp/stp r1, r2, [sp, ip0]

     to avoid mov/movk situations ... for positive values <= 2^24 can add top 12 lsl 12
     then use bottom 12 bits in the unsigned offset field.  Just like add/sub.  If the
     value is larger and masked #FFFF0FFF move top 16 bits lsl 16 then index by the value
     masked by 0FFF.  Otherwise must do mov/movk then index[0]

     For negative indexes can subtrace the negated top 12 bits + 4096 then unsigned index
     with 4095 - negatated value's bottm 12 bits (check that arithmetic!).  Something
